#  Blood Donation Tracking System

This project is a **C++ based Blood Donation Tracking System** developed as part of my early university coursework to practice **object-oriented programming (OOP)** concepts and file-based data management.

The system allows managing **donors, patients, and blood stock** through a structured and modular design.

---

##  Project Scenario

The application simulates a simple blood donation system used in a hospital or blood bank environment.

- **Donors** can register, donate blood, and have their donation history recorded.
- **Patients** can be registered and managed within the system.
- **Administrators** are responsible for managing blood stock, donor information, and patient records.

The system ensures donor eligibility checks and maintains blood group stock levels accordingly.

---

##  Features

- Donor registration and information management
- Blood donation tracking
- Donor eligibility checking
- Blood stock management by blood group
- Patient registration and login
- Admin login and system control
- File-based data storage for donors and patients
